# Workster

Affordable all-in-one business suite for small agencies (10-20 members).

## Stack
- Next.js (App Router, TypeScript)
- Supabase (Postgres, Auth, Storage, RLS)
- Tailwind CSS

## Local development
```bash
npm install
npm run dev
```
